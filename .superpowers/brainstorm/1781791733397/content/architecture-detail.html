关键约束

  • 每次重跑创建新 run_id，不覆盖历史
  • library.db 与 runs.db 分离，降低锁竞争
  • worker 只允许一个 GPU 任务进入 running
  • 所有评分保存原始值与最终值

部署要求

  • 数据库与 FAISS 放在 WSL ext4 Volume
  • 原视频以只读目录挂载到 /media
  • 移除容器内的 Windows 路径和 wsl 命令
  • 宿主 Ollama 需允许 Docker 网桥访问
